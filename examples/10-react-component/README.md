# 10-react-component

## EN

A React + Vite + TypeScript todo list app. The agent's task is to add drag-and-drop sorting using `@dnd-kit/sortable`, install the dependency, update state on reorder, and add tests.

### Structure

- `src/TodoList.tsx` — Static todo list component (starting point)
- `test/TodoList.test.tsx` — Basic rendering tests
- `task.txt` — Task description for the agent

### Run

```bash
./verify.sh
```
